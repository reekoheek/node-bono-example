# node-bono-example

API server example using node version of Bono and Norm.

## Initialize

```sh
git clone https://github.com/reekoheek/node-bono-example.git myproject
cd myproject
rm -rf .git
git init
# and dont forget to modify package.json
```

## How to develop

```sh
npm i
npm run dev
```

## Migration

```sh
npm run migrate
```
